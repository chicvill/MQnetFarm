MQnet Smart Farm

전 세계 어디서든 연결되는 클라우드 기반 지능형 스마트팜 솔루션.
하드웨어부터 클라우드까지, 완벽한 통합 환경을 구축하세요.

ESP32-S3 AI Gateway

시스템의 통신 허브이자 AI 연산 장치입니다.

  • 16MB Flash / 8MB PSRAM 모델 권장
  • S3-BOX 연동으로 음성 인식 및 로컬 AI 수행
  • ESP-NOW 및 WiFi 다중 스택 지원

ESP32-P & Nodes

배터리 구동 및 데이터 수집을 위한 센서 노드입니다.

  • Deep Sleep 최적화로 배터리 수명 연장
  • 초소형 PICO 모델을 활용한 소형 노드 구축
  • Plug-and-Play 방식의 센서 등록 지원

E-TW11 (EW11) Bridge

산업용 장비를 클라우드로 연결하는 이더넷 브리지입니다.

  • RS485/Modbus RTU to Web 프로토콜 변환
  • 고정 IP 및 터널링 모드 지원

GitHub CI/CD

소스 코드 관리 및 자동 배포 시스템입니다.

  • Push 시 Render 클라우드와 자동 동기화
  • Secret Management를 통한 API 키 보호

Docker Container

어디서든 동일하게 동작하는 가상화 실행 환경입니다.

  • OpenCV 및 Python 라이브러리 일관성 보장
  • docker-compose를 이용한 원클릭 환경 구축

Render Hosting

전 세계 수백 개의 농장을 관리하는 서버 인스턴스입니다.

  • 싱가포르 등 글로벌 리전 서버 운영
  • SSL 보안 인증서 및 전용 도메인 제공
